# 💰 AI Financial Advisor & Expense Manager

## 📌 Project Overview
This project is an AI-powered personal finance assistant that automatically extracts expenses from payment screenshots and provides basic financial advice based on spending patterns.

The goal of Week 1 was to build a working system that:
- Extracts transaction details from screenshots using OCR
- Detects transaction amounts automatically
- Categorizes expenses intelligently
- Tracks overall spending
- Provides financial suggestions

## 🔍 Features Implemented (Week 1)

- Screenshot-based expense extraction
- OCR integration using Tesseract
- Amount detection using Regular Expressions
- Automatic category detection
- Manual expense entry
- Expense tracking dashboard
- Category-wise visualization
- Rule-based financial advice
